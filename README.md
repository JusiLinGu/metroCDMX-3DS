# Mapa del metro para Nintendo 3DS
Aplicación de mapa del metro disponible para Nintendo 3DS 

## Instalación
- Se requiere descargar el archivo MetroCDMX.3dsx
- Crear una carpeta llamada 3ds en la raiz de la memoria micro SD
- Dentro de la carpeta anterior crear una carpeta llama metroCDMX
- Guardar el archivo MetroCDMX.3dsx en esta carpeta
- Abrirlo con Homebrew para poder usarla (Tu 3DS debe estar liberada)
